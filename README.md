
# Intranet

## Descripción

Este es el repositorio de la intranet corporativa. Proporciona acceso centralizado a recursos, documentación y herramientas internas para los empleados.

## Características

- Acceso a documentación interna
- Directorio de empleados
- Tablero de anuncios
- Recursos compartidos
- Base de conocimiento

## Requisitos

- Acceso de red corporativo
- Navegador web moderno
- Credenciales de autenticación

## Instalación

```bash
git clone <repository-url>
cd intranet
npm install
npm start
```

## Uso

Accede a la intranet a través de tu navegador en `http://intranet.local`
